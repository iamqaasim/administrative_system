About this project

What type of admin system is this?:

This is a User Administrative System. It allows administrators to create and manage user accounts, assign roles and permissions, and control access to different features and data within the system.

Details:

Users

In this scenario there are 3 types of users:

  • Employee
  • Manager
  • HR Admin (super user)

Employees

Once a new employee is created, a default username and password is generated for them

  • Username: username
  • Password: Password123#
An employee can:
  • only see and edit their own data
  • work for multiple departments

Mangers

A manger is also an employee, so they are created the same way

  • Username: username
  • Password: Password123#
A manager can:
  • only see and edit their own data
  • manage multiple employees
  • manage multiple departments
  • only see employees in their department

HR admins (super users)

A HR admin cannot be created they login with certain credentials to indicate that they are super users

  • Username: hradmin@test.com
  • Password: TestPass1234
A HR admin can:
  • see all employees
  • see all departments
  • change employees data
  • change employees status
  • change department managers
